API Quick Start

Use the endpoints below to access our stock data.

Last 5 Days Data

Fetch closing prices for the last 5 trading days.

http://127.0.0.1:8000/stocks/AAPL

Example usage with `curl`:

curl -X GET "http://127.0.0.1:8000/stocks/AAPL" \ -H "Authorization: Bearer YOUR_API_KEY"

Live Market Data

Get the current, real-time stock price and data.

http://127.0.0.1:8000/stocks/live/AAPL

Example usage with `fetch`:

fetch('http://127.0.0.1:8000/stocks/live/AAPL', { headers: { 'Authorization': 'Bearer YOUR_API_KEY' } }) .then(response => response.json()) .then(data => console.log(data));
